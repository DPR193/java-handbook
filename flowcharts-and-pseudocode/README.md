    ```

      If a > c then
        Print a

        Print c
    ```

## 4. Find If a Number is Prime or Not

**Pseudocode**
1. Start
2. Input n
3. Let div = 2
4. <br> 
  
    ```
    While div < n do
      If n % div == 0 then
        Print "not prime"
        Exit
      Else
        div = div + 1
    ```
5. Print "prime"
6. Exit

## 5. Sum of First N Natural Numbers

**Pseudocode**
1. Start
2. Input n
3. Let val = 1 & sum = 0
4. <br>

   ```
   While val <= n do
     sum = sum + val
     val = val + 1
   ```
5. Print sum
6. Exit

## 9. Flowchart for calculating the average from 25 exam scores.

**Pseudocode**

6. <br>
